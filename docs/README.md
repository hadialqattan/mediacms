# API Documentation

Auto-generated OpenAPI/Swagger documentation for the project APIs.

## Generating Documentation

### Generate All Documentation

```bash
task swagger:gen
```

This generates documentation for both CMS API and Discovery API.

## Accessing Documentation

### Running Services

Start the services:

```bash
task up
```

### Swagger UI

- **CMS API**: http://localhost:8080/swagger/index.html
- **Discovery API**: http://localhost:8081/swagger/index.html
